# Task Tracker

Demo project for api

## Getting Started

This is a Spring Boot application created with:
- Spring Boot 4.0.1
- Java 21
- Maven
- Spring Web

## Running the Application

To run the application, use:

```bash
mvn spring-boot:run
```

Or build and run the JAR:

```bash
mvn clean package
java -jar target/tasktracker-0.0.1-SNAPSHOT.jar
```
